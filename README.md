# UserScripts
This repository contains many different userscripts I have created over the years. Some ranging from simple redirects to full-on reworking functionality of websites.
These were built to work with Chrome/Tampermonkey; they may not work on other environments.

You can also find me on [GreasyFork](https://greasyfork.org/users/2392-daku).

various userscripts I have created for various sites with the simple goal of "making life easier".
All of these were built to work with Chrome+Tampermonkey in mind, so they may not work if not ran under the same environment.

## Status
Over the years website functionality has improved, and also changed to the point where either userscripts are no longer needed, or they have broken.
I have quite a few of these; they can be found in the `@Retired` folder.
